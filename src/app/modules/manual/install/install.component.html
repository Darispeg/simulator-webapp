MANUAL DE INSTALACION
El propósito de este manual es proporcionar una guía detallada que explique todos los pasos necesarios para poner en marcha el "Simulador de patrullaje de reconocimiento de ruta y área para el entrenamiento del comandante de patrulla". A continuación, se presentará un desglose de los tres procesos clave involucrados en la implementación:
  • - Instalación de los componentes de realidad virtual
  • - Topología de conexión
  • - Instalación del simulador.
INSTALACION DE LOS COMPONENTES DE REALIDAD VIRTUAL
Test de Performance
Para llevar a cabo las pruebas de rendimiento de la computadora, se utilizó el software gratuito proporcionado por Steam llamado "Steam performace Test" . Esta herramienta permite realizar un análisis rápido del hardware del sistema y proporciona retroalimentación sobre la capacidad de la computadora para ejecutar aplicaciones de realidad virtual. Pasos a seguir:
  • - Descargar Steam performace Test
  • - Instalar Steam performace Test.
  • - Iniciar pruebas.
  • - Confirmar resultados obtenidos.
Instalación de la aplicación Oculus
Ingresa a la página oficial de Oculus, una vez en el sitio web busca la sección de descargas, puedes encontrarla navegando a través de las opciones o accediendo directamente al siguiente link Oculus Setup Pasos a seguir:
  • - Descargar la aplicacion Oculus Desktop.
  • - Instalar Oculus Desktop
  • - Iniciar la aplicacion.
TOPOLOGÍA DE CONEXIÓN.
Conexión Link.
La conexión Link te permite conectar tus Oculus Quest 2 a una computadora mediante un cable USB para acceder a una experiencia de realidad virtual más potente. Esto te permite jugar juegos y utilizar aplicaciones que requieren un mayor rendimiento gráfico. A continuación, los pasos para establecer la conexión Link. Pasos a seguir:
  • - Necesitarás un cable compatible con USB 3.0
  • - Conecta un extremo del cable a tu Oculus Quest 2 y el otro extremo a un puerto USB 3.0 en tu computadora.
  • - Seleccione el método de conexión Link (cable).
Conexión AirLink.
Los Oculus Quest 2 también admiten la conexión inalámbrica, lo que te permite disfrutar de la realidad virtual sin cables ni necesidad de una computadora. Aquí tienes los pasos para configurar la conexión inalámbrica.
  • - Asegúrate de que tus Oculus Quest 2 y tu teléfono inteligente estén conectados a la misma red Wi-Fi
  • - Descarga e instala la aplicación Oculus en tu teléfono inteligente desde la tienda de aplicaciones correspondiente.
  • - Abre la aplicación Oculus en tu teléfono y sigue las instrucciones para vincular tus Oculus Quest 2.
INSTALACION DEL SIMULADOR
Simulador de patrullaje
Ingrese al enlace Patrol Simulator, este enlace te redireccionara a Google Drive, donde encontrarás una carpeta específica llamada “windows” dentro de tu cuenta. Si no has iniciado sesión en tu cuenta de Google, es posible que se te solicite que lo hagas. Ingresa tus credenciales de inicio de sesión para acceder a la carpeta. Una vez dentro de la carpeta en Google Drive, podrás explorar y descargar los archivos que contiene Pasos a seguir:
  • - Descargar la carpeta llamada Windows.
  • - Descomprime la carpeta
  • - Abre la carpeta y busca el archivo "RPatrolSimulator.exe"
  • - Haz doble clic en el archivo "RPatrolSimulator.exe" para abrirlo
