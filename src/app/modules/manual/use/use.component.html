El objetivo principal de este manual es proporcionar una guía clara y concisa para que el usuario pueda comprender y aprovechar al máximo las capacidades del simulador.
Para lograr esto, se han desglosado las funcionalidades, brindando una explicación detallada de cada una de ellas. Estas funcionalidades son:

INICIO DE SESIÓN
Configuración del sistema guardián
El sistema guardián es una función de seguridad integrada que te permite establecer límites en realidad virtual que aparecerán cuando te acerques demasiado al perímetro de la zona de juego. Pasos a seguir:
  • - Definir zona de juego.
  • - Encender Visores.
  • - Establecer limite del suelo.
  • - Dibujar area de juego.
  • - Asegurar limites del area.

CONTROLES DEL SIMULADOR

Se presenta una imagen que ilustra las funciones de los botones en los controles táctiles de Oculus. Esta imagen proporciona una guía visual detallada de los distintos botones y su funcionalidad en los controles:

  • 1. Stick analógico:
     • Izquierdo (Locomoción): Este stick te permite controlar el movimiento dentro del entorno virtual. Puedes desplazarlo en diferentes direcciones para moverte en el espacio virtual.
     • Derecho (Giro): Este stick te permite girar dentro del entorno virtual. Puedes rotar en el entorno virtual sin girar la cabeza en la vida real.
     • Click: Abre el menú de técnicas de patrullaje, donde podrás seleccionar las diferentes técnicas implementadas dentro del simulador.
  • 2. Botón de menú del sistema:
     • Abre el menú de Oculus que se puede utilizar para configurar algunas funciones como por ejemplo el sistema guardián.
  • 3. Botón “A”:
     • Se utiliza para cambiar la vista entre las interfaces del Mini mapa y la lista de tareas.
  • 4. Botón de disparo (Trigger):
     • Este botón se utiliza para realizar acciones de disparo, como interactuar con elementos específicos del entorno o disparar un arma virtual.
  • 5. Botón de agarre (Grip / Grab):
     • Este botón permite agarrar objetos o realizar acciones de agarre en el entorno virtual. Al presionarlo, podrás sujetar y manipular objetos virtuales con mayor precisión.
MENÚ DE TÉCNICAS DE PATRULLAJE
Menú de técnicas
El menú que facilita la comunicación entre el usuario y los NPC se presenta como una interfaz interactiva que ofrece varias opciones seleccionables. Para elegir una opción específica, simplemente se debe presionar el botón de disparo (trigger) en el control correspondiente a la opción deseada. Al utilizar el botón de disparo (trigger) para seleccionar una opción del menú, se activa la acción correspondiente y se establece la comunicación con el NPC de acuerdo a la opción elegida.

Técnicas de reconocimiento de ruta. Selecciona la opción “RUTA” que abre un panel con las técnicas de reconocimiento de ruta que se pueden realizar.

Estas técnicas se dividen en tres categorías: "Columna", "Cuña" y "Reloj". Cada técnica ofrece una estrategia particular para el reconocimiento de los NPC en el simulador.

  1. Columna: Al seleccionar esta opción los NPC adoptan la formación en columna de tiradores. Cada equipo ocupa las posiciones delimitadas por el reglamento de instrucción de patrullaje.
  2. Cuña: Al seleccionar esta opción los NPC adoptan la formación en cuña. Cada equipo ocupa las posiciones delimitadas por el reglamento de instrucción de patrullaje. Con ayuda del mini mapa se puede ver las formaciones que realizan los NPC.
  3. Reloj: Al seleccionar esta opción los NPC realizan un alto de seguridad “técnica el reloj”. Cada equipo ocupa las posiciones delimitadas por el reglamento de instrucción de patrullaje.

Además de las técnicas de reconocimiento, en la parte inferior del panel se encuentran tres opciones de distancia: "Small", "Medium" y "Large". Estas opciones permiten ajustar la distancia a la cual los NPC realizan la técnica de reconocimiento seleccionada. Al elegir una opción de distancia, se determinará la proximidad entre el usuario y los NPC durante la aplicación de la técnica.

Tecnicas de reconocimiento área

Técnicas de reconocimiento de área. Al seleccionar esta opción, se abre un panel que presenta las diferentes técnicas de reconocimiento de área disponibles. Estas técnicas se dividen en 2 categorías: "Cajón" y "Trébol". Cada técnica ofrece una estrategia particular para el reconocimiento de los NPC en el simulador.

  1. Cajón: Al seleccionar esta opción, el equipo de seguridad conformado por NPC realiza un reconocimiento en cajón.
  2. Trébol: Al elegir esta opción, tanto el equipo de seguridad como el equipo de asalto conformado por los NPC realizarán un reconocimiento en la zona designada como trébol. Durante este reconocimiento, los NPC llevan a cabo una estrategia táctica de exploración en forma de trébol, cubriendo de manera eficiente los puntos clave.
